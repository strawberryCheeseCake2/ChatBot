# ChatBot
Open AI API와 Firebase를 활용한 챗봇 프로젝트 입니다.

# 기능
- 질문 대답
- 오타 수정
- 짧은 이야기 생성
- 다른 유저와 대화
- 구글 로그인을 통한 유저 구분

# 사용한 라이브러리
- Alamofire
- Google Sign In
- Firebase
- SnapKit

# 아키텍처
- MVC 패턴
